判定尺（合計点）

GO

12点以上

買う候補。本線で検討。

HOLD

8〜11点

様子見。近況・残口・予算で再判定。

PASS

7点以下

見送り。今は手を出さない。

同予算プール内では、GOの中でも相対1位を本命にする。

即PASS（1つでも該当で見送り）

満口
一口 4.0万円超
近況に「骨折・跛行・手術・予後不良」

カタログで見る順

  1. 募集状況 GO 募集中 / 残口わずか　 PASS 満口
  2. 母 → 母母（family_mare） GO 広尾独自 S/A または全体牝系 S/A　 HOLD 独自 B/C・全体 B　 PASS 標本不足Uのみ
  3. 予定調教師（広尾実績） GO S/A　 HOLD B　 PASS C/U だけで牝系も弱い
  4. クロス（牝系 × 厩舎） GO クロス S/A は即加点の本命条件
  5. 父 GO コントレイル / カナロア / キタサンブラック / Flightline 等の上位枠
  6. 一口価格 GO ≦2.0万　 HOLD 2.0〜2.5　 ≧2.5 は牝系SかクロスSが欲しい
  7. 近況（あれば） GO 順調・快調　 PASS 骨折・跛行・手術　 HOLD 軽量・短期安静

早見：厩舎・父（正本マップ）

厩舎グレード

  • S 矢作芳人 / 友道康夫 / 須貝尚介 / 池江泰寿
  • A 蛯名正義 / 尾関知人 / 高柳大輔 / 四位洋文
  • B 杉山晴紀 / 武英智 / 荒川義之 / 黒岩陽一 など
  • C 奥村武 / 中尾秀正 / default

父（tier目安）

  • top コントレイル / キタサンブラック / ロードカナロア / Flightline 等（ルール値≧3.5）
  • mid エピファネイア / ナダル / サートゥルナーリア 等（≧2.8）
  • low それ以外・未登録寄り

注: 須貝は芝マイル〜中距離・牝向き。ダート向き牡を預ける型とは別評価（適性一致なら減点しない）。

ポケット短縮カード

  1. 満口 → PASS
  2. 母名 → 母母。広尾独自 S/A or 全体 S/A なら◎
  3. 厩舎が広尾 S/A なら◎ / クロス S/A なら即GO候補
  4. 父が上位 + 一口 ≦2万で加点
  5. 近況に骨折・跛行・手術 → PASS / 順調なら +1
  6. 合計 12〜 GO / 8〜 HOLD / 未満 PASS
  7. 同予算プール内の相対1位を本命に
